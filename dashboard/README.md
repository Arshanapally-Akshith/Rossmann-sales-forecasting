# 📊 Power BI Dashboard

This folder contains the interactive Power BI dashboard for the Rossmann Sales Forecasting project.

## 📁 File

* `rossmann_dashboard.pbix` — Main dashboard file

## 📌 Description

The dashboard provides business-level insights across:

* Executive sales overview and KPIs
* Demand patterns and seasonality
* Model performance (Actual vs Predicted)
* Anomaly detection insights
* Backtesting and evaluation metrics

## ⚠️ Note

GitHub cannot preview `.pbix` files.
Please download and open using **Power BI Desktop**.
